# Abstract
Determining the harvest-ready state of okra (Abelmoschus esculentus) is a time-sensitive agronomic decision whose automation has long been constrained by the architectural separation of fruit localization and maturity staging into independent, sequentially executed modules. We depart from this convention by unifying both tasks within a single anchor-free convolutional detection framework and introduce AgroNet, an end-to-end framework built upon an ImageNet-pretrained ResNet-50 backbone extended with a purpose-designed multi-scale neck and maturity-aware detection heads. Central to our design is the Gated Skip Fusion mechanism, a learnable sigmoid gate that adaptively arbitrates, per sample and per Feature Pyramid Network (FPN) scale, between backbone feature maps and bilinearly-resized raw-image texture patches, enabling the network to selectively exploit pod surface color and epidermis pattern.  Multi-scale feature integration is achieved through a Cross-Scale FPN Neck that combines Query-Key-Value cross-scale attention at the P5→P4 level with Channel-Selective Fusion Token calibration at the P4→P3 level, yielding a Path Aggregation Network-augmented bottom-up path that provides precise bounding-box localization across the full pod size spectrum. Each of the three detection scales is served by a Maturity-Aware Decoupled Head that extends the standard decoupled box-regression and classification branches with a lightweight ordinal branch predicting a continuous ripeness score ∈ [0, 1] per anchor. This branch is jointly trained with an Ordinal Ranking Loss that enforces the pairwise biological ordering constraint s^immature  < s^(under-mature)<s^mature< s^(over-mature), structurally prohibiting stage-order violations during inference, and an Aspect-Ratio Prior Loss that penalizes geometrically implausible detections. At inference, hard non-maximum suppression is replaced by Gaussian Soft Non Maximum Suppression (NMS) with post-NMS cluster-merge deduplication preserves valid detections in dense pod clusters while suppressing near-duplicate predictions. Robust convergence under the class imbalance endemic to small agricultural datasets is achieved through a 5-component hybrid loss complementing CIoU box regression, binary cross-entropy classification and Distribution Focal objectives applied jointly to both detection and maturity branches. Extensive experiments conducted on the OkraFarm benchmark, comprising 526 field images collected across varied illumination conditions, acquisition angles, and phenological stages, demonstrate that AgroNet achieves an accuracy of 97.99 % with competitive detection scores (mAP@50=0.911,mAP@50:95=0.645) alongside accurate four-level maturity discrimination, as corroborated by per-class precision, recall, F1-score, and ROC-AUC metrics. Spatial interpretability is natively provided through class-based activation maps, offering agronomists transparent visual evidence of the cues driving each prediction. These results establish AgroNet as a principled and practically deployable alternative to fragmented multi-stage pipelines for in-field crop maturity monitoring.
